# 🏢 IT 자산 및 비용 자동 분류 관리 시스템

SQLite와 Python(Streamlit)을 사용하여 기존 마스터 데이터의 Description을 신규 데이터와 비교하여
분류값(**제외** / **정보기술** / **정보보호**)을 자동으로 매핑하는 시스템입니다.

---

## 🔧 설치 및 실행

```bash
# 1. 패키지 설치
pip install streamlit pandas rapidfuzz openpyxl

# 2. 폴더 이동 후 실행
cd it_asset_mgmt
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 접속

---

## 📖 사용 순서

### 1단계: 기준정보 등록
- 좌측 **📋 기준정보관리** → 자산 또는 비용 탭
- 엑셀 파일 업로드 (기존 데이터 전체 삭제 후 대체)

### 2단계: 분석 실행
- 좌측 **🔍 분석** → 자산 또는 비용 선택
- **Step 1**: 분석 대상 엑셀 업로드 → "임시 테이블에 저장"
- **Step 2**: 유사도 임계값 조정(기본 85) → "분석 실행"
- 결과: 완전일치/유사매칭/미매칭 통계 + 색상 강조 테이블

### 3단계: 결과 다운로드
- **Step 3**: "엑셀 다운로드" → 제외/정보기술/정보보호가 채워진 엑셀 파일

## 📊 엑셀 컬럼 형식

### 자산 (Asset)
| 엑셀 컬럼명 | 설명 |
|---|---|
| Asset Number | 자산번호 |
| Description | 자산 설명 (매칭 기준) |
| 코스트센터 | 코스트센터 코드 |
| 일련번호 | 시리얼번호 |
| 코스트센터.1 | 코스트센터 보조 |
| 자산클래스내역 | 자산 분류 |
| 연중 상각비 | 감가상각비 |
| **제외** | 분류: O 또는 빈칸 |
| **정보기술** | 분류: O 또는 빈칸 |
| **정보보호** | 분류: O 또는 빈칸 |

### 비용 (Cost)
| 엑셀 컬럼명 | 설명 |
|---|---|
| GL Date | 전기일 |
| 손익 센터 | 손익센터 코드 |
| 코스트 센터 | 코스트센터 코드 |
| Account | 계정 코드 |
| Account Name | 계정명 |
| 사업 영역 | 사업영역 코드 |
| 문서 유형 | 문서 유형 |
| Description | 비용 설명 (매칭 기준) |
| 금액(차변) | 금액 |
| **제외** | 분류: O 또는 빈칸 |
| **정보기술** | 분류: O 또는 빈칸 |
| **정보보호** | 분류: O 또는 빈칸 |

## ⚙️ 매칭 로직
1. **전처리**: Description에서 공백·특수문자 제거 → 대문자 변환
2. **1차 완전일치**: 전처리된 키가 100% 일치 → 분류값 즉시 매핑
3. **2차 유사도 매칭**: `rapidfuzz.token_sort_ratio` 유사도 계산, 임계값 이상 시 매핑
4. **미매칭**: 임계값 미만 → 수동 확인 필요

분류값은 마스터의 **"O"** 값을 그대로 복사합니다.
